# College Website

A comprehensive, responsive college website developed as a digital platform. This repository contains the source code for the website, showcasing the institute's academic programs, administrative structure, student life, and major announcements.

## ✨ Features
- **Responsive Design**: Ensuring a seamless browsing experience across all devices (desktops, tablets, mobile phones).
- **Navigation Menu**: A multi-level dropdown navbar providing easy access to all institute's departments, academics, and administrative sections.
- **Dynamic Carousel**: Hero section image slider showcasing campus life.
- **Announcement & Events Portal**: Quick widgets on the homepage for recent alerts, upcoming events, and news updates.
- **Information Architecture**: Specialized pages for B.Tech, M.Tech, Ph.D. admissions, faculty, staff, library, and alumni sections.

## 🛠 Technologies Used
The project is built entirely on the frontend standard web stack ensuring fast loading times and maintainability:
- **HTML5 & CSS3**: For structuring and styling the web pages.
- **Bootstrap 3.3.6**: For responsive grid layouts and rapid UI component development.
- **JavaScript & jQuery (v2.2.4)**: For DOM manipulation, popovers, and interactive elements.
- **Slick Carousel**: Uninterrupted hero banner slideshows.
- **FontAwesome**: Modern icons used thoroughly across the site.

## 🚀 Setup and Installation
Running the project locally is incredibly straightforward as it's a static site. No complex build tools or backend servers are required.

### Prerequisites
- A modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, etc.)
- (Optional) A local development server extension like *Live Server* for VS Code.

### Steps
1. **Clone the repository:**
   ```bash
   git clone https://github.com/shubhamauti9/college-website.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd college-website
   ```
3. **Run the site locally:**
   - Double-click the `src/index.html` file to open it directly in your browser.
   - *Alternatively*, open the project in VS Code and click on **Go Live** if you have the Live Server extension installed.

## 💻 Usage
The site structure is split into logical fragments mimicking the real-world administrative flow of a college. Start from `src/index.html` and traverse the links from the navigation bar to preview subpages like `btech_cse.html`, `faculty.html`, or `gallery.html`.
